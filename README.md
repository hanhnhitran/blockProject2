# blockProject2
# What your program does
This program checks if a password is weak, medium or strong. It looks at length, letters, numbers, and symbols. It gives a simple score to help users make better passwords.

# How to run the program
1. Open terminal or command prompt.
2. Run: python PasswordStrengthChecker.py.
3. Type a password and see the result.
No extra libraries are needed.

# Warnings/Limitation
- This is a simple school project.
- It is for educational use only.
- It does NOT guarantee your password is fully safe.
- Do NOT use this tool to protect sensitive information.

# Ethical use and responsibility
- This tool is made to help people learn and make stronger passwords.

Possible misuse:
- Someone could change the code and try to use it for bad purposes.
- It may give users a false sense of security if they think a "strong" result means perfect safety.

Responsible use:
- Use it only to check your own passwords.
- Use it for learning basic security ideas.
- Do not use it to hack or collect private information.
